# MealMate
ABOUT -capstone project

          //         MealMate Web App
Description of  MealMate:
=> MealMate is a web application that allows users to search and explore different meal recipes. Users can view meal details such as name, category, and instructions. The main purpose of this project is to practice API integration and JavaScript concepts.

Tools Used :
* HTML
* CSS
* JavaScript
* Fetch API
 
API Information :
This project uses TheMealDB API, which provides data about meals including name, category, image, and recipe instructions.

Main Features
1. Search meals by name
2. Filter meals by category
3. Sort meals
4. View detailed recipe information
